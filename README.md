# 摩托车D照全科考试指南

> 2026年5月新规版 · 苏州增驾适用 · 纯静态页面 · 移动端优先

一个覆盖**科目一到科目四**全部内容的摩托车D照考试指南网站，帮助考生系统复习、顺利通过考试。

## 在线访问

[https://moto.xiaomimimo.com](https://moto.xiaomimimo.com)

## 项目结构

```
├── index.html      首页导航（四科目入口 + 考试概览）
├── common.css      公共样式表（组件库 + 响应式 + 打印）
├── k1.html         科目一：道路交通安全法律、法规和相关知识
├── k2.html         科目二：场地驾驶技能考试
├── k3.html         科目三：道路驾驶技能考试
├── k4.html         科目四：安全文明驾驶常识
└── spec.md         产品规格文档
```

## 各科目内容

### 科目一 — 理论考试

- 50题 | 100分 | 90分及格 | 45分钟
- 10个章节：基础信息、交通标志、交通标线、手势信号、处罚规则、速度规定、让行规则、灯光使用、安全常识、备考计划
- 包含完整的法规数字速记和处罚对照表
- 交通信号标志按颜色和形状分类识别系统

### 科目二 — 场地驾驶技能

- 80分及格
- 5个项目详解：绕桩/桩考、坡道定点停车和起步、单边桥、直角转弯、档位操作
- SVG 场地路线图（绕桩、坡道、单边桥、档位图）
- 每个项目含考试要求、操作步骤、常见失误

### 科目三 — 道路驾驶技能

- 90分及格
- 9个场景：起步/靠边停车、变更车道/超车、路口通行、特殊路段、掉头、夜间行驶
- 扣分标准对照表（100分不合格 / 10分 / 5分）
- SVG 考试路线图

### 科目四 — 安全文明驾驶常识

- 50题 | 100分 | 90分及格 | 45分钟
- 9个章节：恶劣天气、高速公路、紧急处置、急救知识、文明驾驶
- 多选题专项技巧（组合规律、排除法、陷阱选项）
- 图标化安全知识速记

## 技术特点

- **纯静态** — HTML + CSS + 极简 JS，无框架依赖，部署简单
- **组件化 CSS** — 13+ 可复用组件，CSS 变量主题系统
- **每科独立配色** — 科目一红色、科目二蓝色、科目三绿色、科目四橙色
- **响应式设计** — 适配手机到桌面全尺寸
- **打印友好** — @media print 优化，可直接打印复习
- **离线可用** — 不依赖外部 JS 框架，本地打开即用
