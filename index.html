道路交通安全法律法规

驾驶证管理规定

考点关键内容
准驾车型D照可驾驶普通三轮摩托车、普通两轮摩托车（E）、轻便摩托车（F）
实习期取得驾照后12个月为实习期，实习期内不得驾驶营运车辆
记分周期12个月，满分12分；一个周期内记满12分需参加学习并考试
换证规定有效期满前90日内申请换证；6年有效期内每个周期均未满12分，换10年有效期
增驾条件本记分周期和申请前最近一个记分周期内没有记满12分记录

重点法规数字速记

速度相关
没有限速标志的城市道路：最高 30 km/h
没有限速标志的公路：最高 40 km/h
同方向只有1条机动车道的城市道路：最高 50 km/h
同方向只有1条机动车道的公路：最高 70 km/h
高速公路最低限速：60 km/h，最高限速：120 km/h
距离相关
能见度 < 200米 → 车速 ≤ 60，车距 100米以上
能见度 < 100米 → 车速 ≤ 40，车距 50米以上
能见度 < 50米 → 车速 ≤ 20，尽快驶离高速
交叉路口、铁路道口、急弯路、窄路、桥梁：50米内不得停车
公共汽车站、急救站、加油站：30米内不得停车
时间相关
连续驾驶超过 4小时 应停车休息，休息不少于 20分钟
实习期：12个月
换证申请：有效期满前 90日 内

处罚对照表
饮酒驾车 → 暂扣6个月驾照 + 罚款1000-2000元
醉酒驾车 → 吊销驾照 + 5年内不得重新取得 + 追究刑事责任

交通信号
约占25%分值 · 图标题2026年起零容错

交通信号灯

信号含义
绿灯亮准许通行，转弯车辆不得妨碍直行车辆和行人
黄灯亮警示，已越过停止线的车辆可以继续通行
红灯亮禁止通行，右转弯车辆在不妨碍其他车辆和行人时可以通行
绿色箭头灯准许车辆按箭头方向通行
红色叉形灯/箭头禁止车辆通行
黄色闪烁警示，注意瞭望，确认安全后通过

交通标志识别（核心重点）

按颜色判断

红色禁令标志
黄色警告标志
蓝色指示标志
绿色高速公路指路
棕色旅游区标志
橙色道路作业区

按形状判断

正等边三角形（尖朝上）→ 警告标志
倒等边三角形（尖朝下）→ 减速让行
八角形 → 停车让行
圆形 + 红圈红杠 → 禁令标志
圆形 + 蓝底 → 指示标志
方形 → 指路标志 / 辅助标志 / 旅游区标志

高频易混标志对比

标志A标志B区分要点
禁止通行禁止驶入前者白底红圈（禁止一切车辆）；后者红底白横杠（单行道出口）
禁止机动车通行禁止摩托车通行看圈内剪影：机动车剪影 vs 摩托车剪影
注意行人人行横道黄色三角（警告）vs 蓝色方形（指示）
环形交叉环岛行驶黄色三角警告 vs 蓝色圆形指示
傍山险路堤坝路山体图案 vs 水波/船图案
隧道渡口拱形图案 vs 船和波浪图案
注意儿童学校区域黄色三角警告 vs 配合限速标志

交通标线

标线含义
白色虚线同向车道分界，可越线变道
白色实线同向车道分界，不可越线
黄色虚线对向车道分界，保证安全时可越线超车
黄色实线对向车道分界，严禁越线
双黄实线严禁越线或压线
白色/黄色虚实线虚线侧可越线，实线侧不可
网状线禁止停车
导流线按规定的路线行驶，不得压线或越线

安全行车常识
约占20%分值 · 摩托车安全驾驶是重点

摩托车安全驾驶要点

出车前检查
  • 轮胎气压、磨损情况
  • 制动系统（前后刹）
  • 灯光、喇叭、后视镜
  • 机油、燃油量
  • 车牌是否清晰
行驶安全
  • 必须佩戴安全头盔（驾驶人和乘坐人）
  • 不得超员（轻便摩托车不得载人）
  • 不得牵引或被牵引
  • 双手不得离开车把
  • 车把上不得悬挂物品
特殊路况
  • 雨天：制动距离增大，降低车速
  • 雾天：开雾灯和危险报警闪光灯
  • 冰雪路面：低速，避免急转急刹
  • 弯道：减速、鸣号、靠右行驶
  • 山路下坡：不空挡滑行
让行规则
  • 转弯让直行
  • 右方道路来车先行
  • 右转让左转
  • 下坡让上坡
  • 有障碍方让无障碍方

文明驾驶与应急处置
约占15%分值 · 安全意识是核心

文明驾驶原则

核心原则
礼让行人（斑马线前必须减速或停车让行）
不抢道、不加塞、不强行超车
不向车窗外抛洒物品
夜间会车 150米以外 改用近光灯
禁止使用远光灯：照明良好的城市道路、会车时、跟车时
禁止鸣号区域不得鸣号

事故应急处置

事故处理流程
① 停车 → 保护现场
② 造成人身伤亡：立即抢救受伤人员，并迅速报警
③ 未造成人身伤亡，事实清楚：可自行协商处理
④ 开启危险报警闪光灯，设置警告标志
⑤ 乘车人、过往车辆驾驶人、行人应予以协助
报警电话
交通事故报警：122  |  急救电话：120  |  报警电话：110

伤员急救常识

不要随意移动伤员（除非有二次伤害危险）
出血 → 止血、包扎  |  骨折 → 固定，不要试图复位
昏迷 → 保持呼吸道通畅，侧卧位
烧伤 → 冷水冲洗，不要挑破水泡  |  中毒 → 脱离现场，通风

车辆结构与维护常识
约占10%分值 · 基础知识不可忽视

摩托车基本结构

系统说明
发动机动力来源
传动系统链条 / 皮带 / 轴传动
制动系统前制动（通常右手/右脚）+ 后制动（通常左脚/左手）
转向系统方向把
灯光系统大灯、转向灯、刹车灯、牌照灯
仪表速度表、里程表、油量表、转向指示灯

日常维护检查

轮胎：气压、花纹深度、有无裂纹  |  制动：制动液、刹车片磨损
灯光：所有灯光是否正常  |  机油：定期更换，保持适当油位
链条：适当张紧度和润滑  |  冷却液（水冷）：检查液位

答题技巧总结
掌握规律，提高答题准确率

判断题技巧

通常为"正确"
  • 出现"减速""让行""停车观察""安全距离"
  • 出现"不得""禁止""应当"等规范用语
  • 涉及保护行人、非机动车安全
  • 涉及"谨慎驾驶""注意安全"
通常为"错误"
  • 出现"可以""允许"做违规行为
  • 降低安全标准（如"可以不戴头盔"）
  • "加速通过""迅速通过"等冒险行为
  • 与法规明文规定相矛盾的表述

选择题技巧

四大方法
排除法 — 先排除明显错误选项，缩小范围
关键词法 — "最高"选限制速度 / "最低"选保障安全 / "不得"选最严格限制
安全第一原则 — 涉及"礼让""减速""停车"的选项优先选
处罚题规律 — 罚款越大越严重 / "吊销"比"暂扣"严重 / "终生不得重新取得"只对应逃逸构成犯罪
